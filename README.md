# 🏠 HomeHaven Rental

A full-stack rental platform for home furniture and appliances, allowing users to browse, rent, and manage products through a structured system.

---

## 🚀 Overview

HomeHaven is designed to simplify the process of renting household items by providing an intuitive interface and a backend system for managing products and user interactions.

---

## ⚙️ Features

### 🛍️ Product Management
- Browse available furniture and appliances  
- View product details and availability  

### 👤 User Interaction
- User-friendly interface for exploring items  
- Structured flow for selecting and renting products  

### 🗄️ Data Management
- Stores product and user data efficiently  
- Handles rental-related operations  

---

## 🧱 Tech Stack

- Frontend: React  
- Backend: Java  
- Database: SQL  
- Environment: Linux
